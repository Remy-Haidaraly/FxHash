
# FxHash V.1.0.0

This is application created in python with customtkinter. This application allows to break a hash with a dictionary attack.

list of compatabile hash algorithms : 
- MD5
- SHA1
- SHA224
- SHA256
- SHA384
- SHA512
- SHA3_224
- SHA3_256
- SHA3_384
- SHA3_512
